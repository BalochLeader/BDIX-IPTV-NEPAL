## 🚀 Overview
**BDIX IPTV NEPAL** is a high-performance, automated IPTV scraper that fetches, merges, and delivers a complete M3U playlist. Designed specifically for users in Bangladesh and Nepal, it leverages BDIX-connected servers for lightning-fast streaming.

## 🌟 Key Features
- **🔄 24/7 Automation**: Powered by GitHub Actions, the playlist updates every hour.
- **⚡ BDIX Connected**: Optimized for low-latency streaming on BDIX-supported ISPs.
- **📂 Full M3U Playlist**: A single, comprehensive file for all your favorite channels.
- **🛠️ Zero Maintenance**: Once set up, the bot handles everything automatically.

## 🛠️ How to Use
Copy the link below and paste it into any IPTV player (VLC, MX Player, OTT Navigator, etc.):

```text
https://raw.githubusercontent.com/BalochLeader/BDIX-IPTV-NEPAL/main/playlist.m3u
```

## ⚙️ How it Works
1.  **Scraping**: The Python script (`scraper.py`) scans multiple high-quality IPTV repositories.
2.  **Merging**: It combines the discovered links into a single, clean M3U format.
3.  **Updating**: GitHub Actions runs the scraper on a schedule and pushes the latest version to this repo.
