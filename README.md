
# BDIX IPTV NEPAL

This repository provides an automated IPTV M3U playlist that scrapes links from [Mrgify-BDIX-IPTV](https://github.com/abusaeeidx/Mrgify-BDIX-IPTV) and other sources.

## Features

- **Automated Updates**: The playlist is updated every hour using GitHub Actions.
- **BDIX Support**: Includes BDIX-connected channels for fast streaming in Bangladesh and Nepal.
- **M3U Playlist**: The generated `playlist.m3u` file can be used in any IPTV player.

## How to Use

To use this playlist, copy the following URL and paste it into your IPTV player:

```
https://raw.githubusercontent.com/BalochLeader/BDIX-IPTV-NEPAL/main/playlist.m3u
```

This repository does not host any content. It aggregates publicly available data for informational purposes only. Use at your own risk.
